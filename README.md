# 代维

#### 作品简介
个人网站采取的简约风，大多使用的色块来构建。模块包含了头部，导航，关于自己的介绍，作品，联系和脚注。（字号：25；
颜色：多数是灰色，点击变化的颜色有紫色，粉色，黄色等。）
